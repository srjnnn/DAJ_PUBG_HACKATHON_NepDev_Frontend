# mero-sathi 


**Anonymous Audio Peer Support for Mental Health in Nepal**

---

## 🚀 Overview
A mobile-first platform connecting individuals to **trained Community Mental Health Volunteers (CMHVs)** via **anonymous audio group chats**.  

**Target Groups:** Students, Migrants’ Wives, Earthquake Survivors

**Key Value:** Low-bandwidth, culturally aligned, and volunteer-driven.

---

## 🛠 Hackathon MVP
- **Tech:** WhatsApp / Telegram audio + lightweight web dashboard  
- **User Flow:** Sign Up / Login → Join Peer Group → Anonymous Audio Session → Feedback  
- **Volunteer Flow:** CMHVs manage sessions, moderate chats, submit reports  

---

## 🌱 Scalability Plan
- **Volunteer Growth:** 1-hour training → each CMHV runs 2 sessions/week → 500 volunteers in 1 month  
- **Session Expansion:** Multiple categories + languages (Nepali, Maithili, Tamang…)  
- **Low Cost:** Uses existing apps; minimal backend hosting  
- **Impact Potential:** 500+ CMHVs → 5,000+ participants weekly in 12 months
